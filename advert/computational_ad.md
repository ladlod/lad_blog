# 计算广告学习笔记
## 一、广告
### 广告是什么
>> **广告**：是由已确定的出资⼈通过各种媒介进⾏的有关产品（商品、服务和观点）的，通常是有偿的、有组织的、综合的、劝服性的⾮⼈员的信息传播活动，可作为商业化的⽅式之⼀。
>> **商业化**：是将流量、数据、影响⼒通过商业产品的⽅式转为收⼊的过程。
>> **广告参与方**：⼴告主、媒体（⼴告平台）、受众
### 广告的投放过程可以看做三方博弈的过程
>> **广告主**：广告需求方，主动参与 **目标**：高roi，更低的成本获取更高的收益
>> **媒体**：流量供给方，主动参与 **目标**：流量收益最大化，平衡收益与用户体验
>> **受众**：广告接受者，被动参与
